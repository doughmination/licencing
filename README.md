**Version:** ESAL-1.5
**Author:** Clove Nytrix Doughmination Twilight
**Copyright:** © 2026

The **Estrogen Source-Available Licence (ESAL)** is a custom **source-available software licence** created by Clove Nytrix Doughmination Twilight.

It allows inspection, modification, and redistribution of source code for **non-commercial purposes**, while protecting attribution, trademarks, and commercial licensing rights.

The complete legal licence text is available in the `LICENSE.md` file.

### `derivative-licence-permissions/`

This directory contains records of **written permissions granted for the creation of derivative licences** based on ESAL.

Under **Section 13 (Derivative Licences)** of ESAL, creation of licences derived from or based on ESAL requires **prior written permission** from Clove Nytrix Doughmination Twilight.

Documents in this directory may include:

* written authorisations
* records of approved derivative licences
* attribution requirements
* additional conditions attached to derivative permissions

These documents serve as **public records of authorised derivative licence usage**.

---

### `licence-archive/`

This directory contains **archived versions of the Estrogen Source-Available Licence**.

Archived versions are kept for:

* historical reference
* compatibility with projects using older versions
* transparency regarding licence changes

Projects may continue using the version of ESAL under which they were originally released unless otherwise specified.

Archived files may include versions such as:

* ESAL-1.2
* ESAL-1.3
